# Secure-To-Do-List
A RESTful API backend for a To-Do List made using Python through FastAPI

Set Up Instructions:
- Download the project files
- Set up the Python environment to create a .venv file
- Using pip, install all dependencies
- Database is already set up by default
- Run the project through any compiler such as VSCode, and open a webpage with 127.0.0.1:8000
- Go to 127.0.0.1:8000/docs#/ to see the API documentation
